\section{Описание используемых типов данных}
При реализации алгоритмов будут использованы следующие типы данных:
\begin{enumerate}[label=\arabic*), itemindent=1em]
	\item словарь;
	\item список ключей;
	\item длина массива/словаря --- целое число типа int.
\end{enumerate}
